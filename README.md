## Plan: anchor sets + flex time

- 2 must-sees per night, max
- 30-min buffer between anchors (walks are 10–15 min)
- Arrive 9pm, not 7pm
- Night 1 = lightest plan
- 1 "explore" stage per night (new genre)
- Sit + eat for an hour around 1–2am
